 # Physics Master’s Thesis Template – University of Saïda

This repository contains a LaTeX template for a Master's thesis in Physics, adapted for the **University of Saïda**.  

## About

This template is adapted from the [USTHB Thesis Template](https://www.overleaf.com/latex/templates/usthb-thesis/jwbcnbcgrkdq) created by Dr. Yacine Bouali (Faculty of Electrical Engineering, USTHB) and is licensed under **Creative Commons CC BY 4.0**.  

I have used this template as the basis for my own thesis, making extensive modifications to formatting, structure, and content to suit the specific requirements of the Physics department at the University of Saïda, including additional sections and adjustments to layout.

## Features
- Complete template for Master's thesis in Physics
- Includes sections for abstract, dedication, acknowledgments, table of contents, figures, and tables
- Custom formatting suitable for University of Saïda requirements
- Easy to compile with Overleaf or locally with LaTeX

## Usage

1. Clone or download this repository.
2. Open the main `.tex` file in Overleaf or your LaTeX editor.
3. Replace placeholder text with your own content.
4. Compile to generate the PDF thesis.
